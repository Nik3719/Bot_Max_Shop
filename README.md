# Бот-Магазин MAX

Бот для мессенджера MAX, реализующий функционал интернет-магазина с панелью администратора и интеграцией с Google Sheets.

## Основной функционал

* **Для пользователей:**
  * Просмотр каталога товаров с удобной пагинацией.
  * Добавление товаров в корзину и оформление заказа.
  * Сохранение контактных данных.
* **Для администраторов:**
  * Просмотр списка всех заказов.
  * Принятие заказов в обработку.
  * Синхронизация новых заказов с таблицей Google Sheets.
  * Автоматическая синхронизация по расписанию.

## Технологии

* [maxapi](https://pypi.org/project/maxapi/) — библиотека для работы с API мессенджера MAX.
* **SQLite + aiosqlite** — локальная база данных.
* **gspread + google-auth** — интеграция с Google Sheets.
* **apscheduler** — планировщик задач для автосинхронизации.

## Установка и запуск

1. **Склонируйте репозиторий:**

   ```bash
   git clone https://github.com/Nik3719/Bot_Max.git
   cd ./Bot_Max_Shop
   ```
2. **Создайте и активируйте виртуальное окружение:**

   ```bash
   python3 -m venv venv
   source venv/bin/activate
   ```
3. **Установите зависимости:**

   ```bash
   pip install -r requirements.txt
   ```
4. **Настройте переменные окружения:**
   Скопируйте шаблон конфигурации и переименуйте его:

   ```bash
   cp .env.example .env
   ```

   Отредактируйте файл `.env`, подставив свои значения:

   ```env
   BOT_TOKEN=ваш_токен_бота
   BOT_LINK=ссылка_на_бота

   # ID администраторов через запятую
   ADMIN_IDS=123456789,987654321

   # Интеграция с Google Sheets
   SPREADSHEET_ID=ID_вашей_таблицы
   GOOGLE_CREDENTIALS_FILE=credentials.json
   SYNC_INTERVAL_HOURS=24
   ```
5. **Настройте Google Sheets:**

   * Создайте сервисный аккаунт Google Cloud и скачайте JSON-файл ключей.
   * Переименуйте файл в `credentials.json` (или как указано в `.env`) и поместите в корень проекта.
   * Откройте доступ к вашей таблице Google Sheets (с правами редактора) для email-адреса сервисного аккаунта, указанного в файле ключей.
6. **Запустите бота:**

   ```bash
   python main.py
   ```
