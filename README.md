# Бот-Магазин MAX

Бот-магазин для мессенджера MAX с корзиной, панелью администратора и автоматической выгрузкой заказов в Google Таблицы.

## Установка и запуск

1. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Настройте конфиг:**
   Скопируйте пример настроек и впишите свои данные (токен бота, ID админов, ID таблицы):
   ```bash
   cp .env.example .env
   ```

3. **Подключите Google Sheets:**
   Положите файл `credentials.json` (ключ от Google Cloud) в папку с ботом и выдайте этому сервисному аккаунту права редактора в вашей таблице.

4. **Запустите бота:**
   ```bash
   python main.py
   ```
